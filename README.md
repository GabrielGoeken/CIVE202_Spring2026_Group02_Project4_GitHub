# Project-4 -- Interpreting Risk Levels
The purpose of this project is to evaluate the National Risk Index (NRI) developed by the Federal Emergency Management Agency and determine whether its method for calculating disaster risk introduces bias across different communities. The objective is to perform a sensitivity analysis by creating an alternative definition of risk and comparing it to the existing NRI model. This involves analyzing how changes in the inputs and structure of the risk equation affect outcomes at the Census tract level. By comparing both approaches, the project aims to identify any categorical bias and assess how these differences could impact community risk rankings and future resource allocation. Ultimately, the analysis helps determine whether the current NRI methodology fairly represents risk or if improvements are needed. (NOTE: NRI Raw data files were too large to import into GitHub)
